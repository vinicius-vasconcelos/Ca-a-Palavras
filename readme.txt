regras:
-o jogo tem 7 arquivos prontos(quem tem que mudar o caminho da pasta sempre que iniciar o jogo) e uma opção "Criar".
-"Criar" -  pode criar um arquivo com suas palavras para jogar uma rodada(o arquivo é destruido todo fim de jogo), sempre que entrar no meu "Criar" tem que ir até o fim(não tem a opção de sair antes de termina-lo).

Criação de arquivo:
- O arquivo do usuario deve conter no minimo 5 palavras validas(sem se repetir ou maiores de 6 caracteres), tudo validado.

obs:
